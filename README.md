# ip_address_checker

This is an app to check the ip address.

## How to build

```
$ flutter build apk
or
$ flutter build apk --debug
```

## How to debug on local

### Android

```
$ flutter emulators --launch Pixel_7
$ flutter devices
$ flutter run -d emulator-5554
```
